# admin-dashboard
An admin dashboard
modern, high-performance administrative dashboard built with the MERN Stack (MongoDB, Express, React, Node.js). This project features a secure authentication system, real-time financial tracking, and a sleek UI powered by Tailwind CSS v4.

 Key Features
Secure Authentication: Role-based access control (RBAC) using JWT and Bcrypt for Super Admin login.

Real-time Analytics: Live calculation of revenue, user metrics, and growth percentages with fallback logic for zero-data states.

Global API Handling: Centralized Axios instance with request interceptors to automatically inject JWT tokens into headers.

User Management: Dynamic tables for monitoring users and inviting new members via integrated email logic.
