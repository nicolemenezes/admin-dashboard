An Admin Dashboard
A high-performance MERN Stack (MongoDB, Express, React, Node.js) administrative dashboard featuring Tailwind CSS v4, secure JWT authentication, and real-time financial tracking.

Key Features
Secure Authentication: Role-based access control (Super Admin) using JWT and Bcrypt.

Live Statistics: Real-time calculation of revenue, growth percentages, and user metrics.

User Invitation System: Automated user onboarding via the Resend API.

Financial Tracking: Full CRUD operations for managing business transactions.

Tech Stack
Frontend: React, Vite, Tailwind CSS, Lucide Icons, Axios.

Backend: Node.js, Express.js.

Database: MongoDB Atlas.

Email: Resend API.
